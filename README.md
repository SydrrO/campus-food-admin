# 赛杜甄选管理后台

校内订餐系统，包含 FastAPI 后端、管理后台前端和小程序端。

版本：v4.5

## 项目结构

```
campus-food/
├── backend/                   # FastAPI 后端
│   ├── app/
│   │   ├── api/v1/            # API 路由（用户、订单、菜单、支付、对账、优惠券、管理端）
│   │   ├── core/              # 配置（JWT、微信支付、数据库、Redis）
│   │   ├── db/                # 数据库连接与会话
│   │   ├── models/            # SQLAlchemy 模型（订单、用户、地址、菜品、优惠券、会员）
│   │   ├── schemas/           # Pydantic 数据校验
│   │   ├── services/          # 业务层（超时关单、会员积分、微信支付）
│   │   ├── utils/             # 工具（时间、订单编号生成）
│   │   └── main.py            # 应用入口
│   ├── scripts/               # 运维脚本（初始化数据库、超时关单、数据迁移）
│   ├── certs/                 # 微信支付证书
│   ├── deploy/                # 部署配置
│   └── requirements.txt       # Python 依赖
├── admin/                     # 管理后台前端（Vue 3 + Vant UI，无构建步骤）
│   ├── index.html             # 主页面：订单处理 / 用户数据 / 对账中心 / 菜单管理
│   ├── coupons.html           # 优惠券管理
│   ├── receipt-print.html     # 小票打印页
│   ├── login.html             # 登录页
│   ├── js/
│   │   ├── workbench.js       # 核心业务逻辑（订单、配送、用户、菜单、对账、打印）
│   │   └── api.js             # 后端 API 封装
│   └── css/
│       └── workbench.css      # 样式
├── data/                      # SQLite 数据库文件（生产环境）
├── uploads/                   # 上传文件（小程序素材、头像）
└── backups/                   # 历史备份
```

## 功能模块

### 管理后台

| 模块 | 说明 |
|------|------|
| 订单处理 | 按状态归类展示，支持筛选（关键词、配送日期、餐别、状态），一键状态流转（确认→备餐→配送→完成），卡片标注优惠券使用情况 |
| 商家回执 | 按配送日期/餐别汇总订单，按商家代码分组输出，用于分拣配送 |
| 用户数据 | 用户列表、消费记录、会员状态、积分查询，支持零填充ID搜索（如 0001） |
| 对账中心 | 按日期/餐别统计流水、成本、毛利，商品成本来自菜单维护 |
| 菜单管理 | 分类增删改、菜品增删改/上下架/售罄，一键售罄/补货，商家代码管理 |
| 优惠券 | 发放和删除优惠券 |
| 小票打印 | 支持单张小票和 A4 批量打印，订单记录文本复制 |

### 配送区域

系统预设 8 个配送区域，用户下单时从中选择：男寝、女寝、书4、书5、男9、女9、男7、东区。

### 后端 API

- 用户端：注册/登录、地址管理、下单、支付、优惠券兑换、积分
- 管理端：订单监控、状态流转、菜单维护、用户查询、对账统计
- 微信支付：小程序真实支付 + mock 模式，回调验签

## 会员等级体系 (v4.0)

五级会员制，消费达标自动升级：

| 等级 | 名称 | 消费门槛 | 主题色 | 专属权益 |
|------|------|----------|--------|----------|
| LV1 | 黑铁会员 | ¥0 | 铁灰 `#7B7B7B` | 积分兑券 · 周六95折 · 免配送费 |
| LV2 | 白银会员 | ¥100 | 银色 `#9BA3AE` | + 双倍积分加速 |
| LV3 | 黄金会员 | ¥250 | 金色 `#D4A017` | + 三倍积分加速 |
| LV4 | 铂金会员 | ¥550 | 绿松石 `#4DB6AC` | + 专属客服优先接入 |
| LV5 | 钻石会员 | ¥800 | 靛蓝 `#6C8EE0` | + 五倍积分 + 年度礼盒 |

### 积分规则

- 每次消费可获得消费金额 100 倍的积分
- 邀请好友开通会员，邀请人获得 3000 积分
- 积分兑换券：3000分兑0.5元券、6000分兑1元券、10000分兑2元券
- 仅会员可使用积分兑换

### 会员中心

- 等级专属 Hero 区（渐变遮罩、毛玻璃效果）
- 等级体系进度条（当前等级脉冲光环动画）
- 等级渐变金属字特效（`background-clip: text`）
- 权益卡片（红/绿/蓝三色区分折扣/免配/兑换）
- 底部规则弹窗（各等级权益对照）

## 技术栈

| 层 | 技术 |

## 开发环境

### 后端

```bash
cd backend
python -m venv .venv
.venv\Scripts\pip install -r requirements.txt
cp .env.example .env
# 本地快速启动可配：DATABASE_URL_OVERRIDE=sqlite:///./campus_food.db
.venv\Scripts\python scripts/init_db.py
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

访问 http://localhost:8000/docs 查看 Swagger 文档。

### 管理后台

无需构建，直接用任意 HTTP 服务托管 `admin/` 目录即可。开发时后端已内置静态文件挂载，访问 `/admin` 路径。

### 小程序端

位于独立仓库 `campus-food-wechat/miniapp/`，微信开发者工具导入即可。

### 开发工具

- **Dashboard**: `python scripts\dev-dashboard.py` → http://localhost:8765
  - 实时查看/编辑测试用户数据（会员状态、积分、消费额、昵称）
  - 一键升级最新用户为会员
  - 删除用户（级联清除关联数据）
  - Ctrl+S 保存修改
- **用户管理 CLI**: `python scripts\dev-user-mgr.py last` 快速将最新用户设为会员

### 开发环境配置

- 开发服 `APP_ENV=development` 固定登录用户 #22，避免每次编译创建新账号
- 生产服 `APP_ENV=production` 走正常微信登录流程
- 前端自动静默登录仅限开发环境（API 地址含 `dev-api`）
- 结算页自动选取最大面额优惠券
- 缓存的优惠券校验归属用户，换账号自动清除重选

## 部署

当前生产环境运行在阿里云 ECS 上：

- 后端：两个 uvicorn 实例（端口 8000 生产、8001 开发）
- 前端：Nginx 反向代理，SSH 复用端口 80
- 数据库：SQLite（`/opt/campus-food/data/campus_food.db`）
- 开发库：`/opt/campus-food-dev/data/campus_food_dev.db`

部署脚本见 `backend/deploy.sh`，上传脚本见 `backend/upload-to-server.sh`。
